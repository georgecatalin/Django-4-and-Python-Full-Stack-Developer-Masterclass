VARIABLES.HTML

This is example how to use Django Template Language to fill in dynamic shit on the web page:

First Name = {{first_name | lower }}

Last Name = {{last_name | upper}}

A list = {{my_list}}

An element of the list = {{my_list.1}}

A dictionary = {{my_dictionary}}

An element of a dictionary = {{my_dictionary.inside_key}}

A comment inside Jinja = {# This is a comment #}

    {% for item in my_list %}
  • {{item}}
    • {% endfor %}
    {% for key,value in my_dictionary.items %}
  • {{key}} : {{value}}
    • {% endfor %}
    {% for year in my_list %} {% if year == 1953 %}
  • Anul de nastere al lui tata
    • {% elif year == 1954 %}
  • Anul de nastere al mamei
    • {% else %}
  • {{year}}
    • {% endif %} {% endfor %}
